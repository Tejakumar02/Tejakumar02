## ⚡ Experience

### 🏭 AI Engineer Intern — Defect Scanner &nbsp; `Jul 2025 – Jan 2026`

- 🎯 Deployed YOLO defect detection on **17K+ images** → **mAP50: 0.92**, **13+ FPS** in production
- 🔩 Engineered **spatial-temporal validation** (centroid tracking + ROI checks) for ~95% process compliance (~60 units/shift)
- 🚨 Built automated alert system (tower lamp + buzzer) for real-time line enforcement; reduced manual inspection by **~40%**
- 🐳 Deployed as **Flask REST API** with Docker-ready architecture; full pipeline from CVAT annotation (90K+) → model → monitoring

### 🧠 DL Engineer Intern — FDAI &nbsp; `Dec 2024 – Jun 2025`

- 🔍 Built **RAG pipelines** with LangChain + ChromaDB + locally hosted LLMs (Ollama: Mistral, LLaMA)
- 📄 Integrated **Hugging Face Transformers**, PaddleOCR, and Tesseract for multimodal document understanding (text, tables, entity recognition)
- 🔒 Deployed AI in **air-gapped on-premises environments** — zero cloud dependency
- ⚙️ Built **FastAPI demo apps** exposing model capabilities as RESTful endpoints for stakeholders
